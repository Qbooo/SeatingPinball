# 자리 배정 핀볼

이름 목록을 입력하면 핀볼 게임으로 자리를 배정해주는 웹 프로그램입니다.
Space Cadet 스타일 핀볼 안에서 공들이 튕기다가 구멍에 빠지는 순서대로 자리가 정해집니다.

## 데모

**[바로 실행해보기](https://qbooo.github.io/SeatingPinball)**

## 주요 기능

- **이름 입력** — 쉼표, 스페이스, 엔터로 구분해서 입력
- **자리 제외** — 자리 박스 클릭 → 빨간색, 배정 제외
- **고정 배정** — 자리 박스 안에 이름 직접 입력 → 핀볼 결과와 무관하게 고정
- **핀볼 게임** — 5초마다 5명씩 자동 발사, 범퍼·킥커·플리퍼로 튕기다가 구멍에 빠지면 배정
- **화면 전환** — 핀볼 화면 ↔ 자리 배치표 클릭으로 크기 전환

## 자리 배치 (총 32자리)

```
왼쪽 (16자리)              오른쪽 (16자리)
┌────┬────┬────┬────┐     ┌────┬────┬────┬────┐
│ 25 │ 26 │ 27 │ 28 │     │ 29 │ 30 │ 31 │ 32 │  ← 행1
├────┼────┼────┼────┤     ├────┼────┼────┼────┤
│ 17 │ 18 │ 19 │ 20 │     │ 21 │ 22 │ 23 │ 24 │  ← 행2
├────┼────┼────┼────┤     ├────┼────┼────┼────┤
│  9 │ 10 │ 11 │ 12 │     │ 13 │ 14 │ 15 │ 16 │  ← 행3
├────┼────┼────┼────┤     ├────┼────┼────┼────┤
│  1 │  2 │  3 │  4 │     │  5 │  6 │  7 │  8 │  ← 행4
└────┴────┴────┴────┘     └────┴────┴────┴────┘
                  [ 강사 ]
```

번호 순서: 아래 → 위, 왼쪽 → 오른쪽 (같은 행끼리 묶음)

## 사용법

1. 이름 입력창에 참여자 이름 입력
2. (선택) 제외할 자리 클릭 → 빨간색 표시
3. (선택) 고정할 자리에 이름 직접 입력 → 초록색 표시
4. **게임 시작** 버튼 클릭
5. 공이 구멍에 빠지는 순서대로 자리 배정 완료
